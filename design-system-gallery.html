1.8 — Motion & Interaction

Easing Curves

ease-out

cubic-bezier(0.0, 0.0, 0.2, 1)

Elements entering. Fast start, soft landing. Default for reveals and hover enter.

ease-in

cubic-bezier(0.4, 0.0, 1, 1)

Elements exiting. Smooth start, fast exit. Dismiss and close transitions.

ease-in-out

cubic-bezier(0.4, 0.0, 0.2, 1)

Visible-to-visible state changes. Position shifts and reorder animations.

spring

cubic-bezier(0.34, 1.56, 0.64, 1)

Slight overshoot. Used sparingly on interactive moments — hover lifts, CTA press.

Duration Scale

100msMicro
150msFast
200msDefault
300msMedium
350msEnter
500msEmphasis

100–150ms: hover color changes. 200ms: button press, card lift. 350ms: scroll-triggered reveals. 500ms: hero animations only. Always respect prefers-reduced-motion — disable all decorative transitions.
